# 🎂 Age Classifier

A Python program that classifies a person as a **minor**, **adult**, or **retiree** based on their name and age, provided via the terminal.

---

## 📁 Project Structure

```
Practice 1/
├── main.py              # Core application logic
├── conftest.py          # pytest configuration (sys.path)
├── pyproject.toml       # Project dependencies and configuration
├── uv.lock              # Lockfile generated by uv
└── Unit Tests/
    └── test_main.py     # pytest test suite (27 tests)
```

---

## ⚙️ Prerequisites

- [uv](https://docs.astral.sh/uv/getting-started/installation/) installed globally
- Python 3.10+

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
uv sync --group dev
```

> Creates the `.venv` virtual environment and installs all dependencies, including `pytest`.

---

### 2. Run the application

```bash
uv run main.py
```

**Example output:**

```
--- Age Classifier ---

Enter your name: Maria
Enter your age: 72

Hello, Maria! Since you are 72 years old, you are a retiree.
```

## 🧪 Tests

The test suite covers boundary values, guard clauses, and integration — **27 tests in total**.

### Run all tests

```bash
uv run --group dev pytest
```

### Verbose output (recommended)

```bash
uv run --group dev pytest -v
```

### Quiet output (ideal for CI)

```bash
uv run --group dev pytest -q
```

### Filter by test class

```bash
# Only classify_age tests
uv run --group dev pytest -v -k "TestClassifyAge"

# Only format_message tests
uv run --group dev pytest -v -k "TestFormatMessage"

# Only integration tests
uv run --group dev pytest -v -k "TestIntegration"
```

### Stop on first failure

```bash
uv run --group dev pytest -x
```

## 🏗️ Architecture

| Function | Description |
|----------|-------------|
| `classify_age(age)` | Classifies the age; pure, no I/O |
| `format_message(name, age, classification)` | Formats the final output message |
| `get_validated_age(prompt)` | Validates user input from terminal |
| `main()` | Entry point, orchestrates the flow |

### Business Rules

| Age Range | Classification |
|-----------|----------------|
| 0 – 17 | Minor |
| 18 – 65 | Adult |
| 66+ | Retiree |
